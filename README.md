## 📌 How to Play

- Arrows will appear one at a time (⬆️ ⬇️ ⬅️ ➡️).
- Press the **matching arrow key** quickly.
- Score increases on each correct input.
- Wrong input or timeout ends the game.


2. Open `index.html` in your browser.
